Tipos de selectores que entran en examen

Clase (class=””)

El selector de clase se utiliza para seleccionar elementos que comparten un atributo class específico. Se antepone un punto (.) al nombre de la clase. Por ejemplo, .mi-clase selecciona todos los elementos con class="mi-clase". Este selector es útil para aplicar estilos a múltiples elementos que comparten una misma clase.

Identificador (id=””)

El selector de identificador selecciona un elemento con un atributo id específico. Se antepone una almohadilla (#) al valor del id. Por ejemplo, #mi-id selecciona el elemento con id="mi-id". Dado que los identificadores deben ser únicos en un documento, este selector se utiliza para aplicar estilos a un solo elemento.

Todos los elementos (*)

El selector universal (*) selecciona todos los elementos de un documento. Es útil para aplicar un estilo global a todos los elementos, como establecer un margen o un padding por defecto.

Varios elementos ( p, em, h1, h2)

Este selector selecciona varios tipos de elementos especificados, separados por comas. Por ejemplo, p, em, h1, h2 selecciona todos los elementos p, h1 y h2 h2 h1 . Es útil para aplicar el mismo estilo a diferentes tipos de elementos.

Elementos contenidos en otros elementos (ol li)

El selector de descendiente selecciona elementos que son descendientes de otros elementos. Por ejemplo, ol li selecciona todos los elementos li que están dentro de un ol. Es útil para aplicar estilos a elementos en una jerarquía específica.

Sobrevuleo (p:hover)

El selector de pseudo-clase :hover selecciona un elemento cuando el puntero del mouse pasa sobre él. Por ejemplo, p:hover aplica estilos a un párrafo cuando el usuario pasa el mouse sobre él. Es comúnmente utilizado para efectos interactivos, como cambiar el color de un enlace al pasar el mouse sobre él.
